# 건축 음향 규격 — 층간소음 기준 및 흡음 등급

## 개요

한국 층간소음 및 건축 음향 관련 규격의 종합 정리. 제품명세(α_w, L_n,w)와 시장 진입 기준선을 정의.

## ISO 11654 — 흡음 등급 (Sound Absorption Class)

| Class | α_w 범위 | 용도 예시 |
|:-----|:--------:|:---------|
| **A** | 0.90~1.00 | 공연장, 강당, 녹음실 |
| **B** | 0.80~0.85 | 사무실, 교실, 회의실 |
| **C** | 0.60~0.75 | 복도, 로비, 상업공간 |
| **D** | 0.30~0.55 | 소매점, 경공업 |
| **E** | 0.15~0.25 | 기본 수준 |
| 미분류 | < 0.15 | 흡음성 거의 없음 |

- α_w 측정: ISO 354 잔향실, 1/3 옥타브 21개 주파수 평균
- 저주파 Shape indicator: α_w(N) = 저주파 부족, (M) = 중간 우세, (H) = 고주파 우세
- **폐섬유 흡음재 목표: α_w ≥ 0.80 (Class B)** — 100mm 두께 기준

## KS F 2862 — 바닥충격음 등급

### 중량충격음 (Heavy-weight, Bang machine / Rubber ball)

| 등급 | L_iA,Fmax (dB) | 평가 | 시장 의미 |
|:---:|:-------------:|:----:|:--------|
| **1등급** | ≤ 43 | 우수 | 프리미엄 (건축비 +2%) |
| **2등급** | 44~48 | 양호 | 고급 아파트 |
| **3등급** | 49~52 | **법적 기준** | 최소 요구사항 |
| 4등급 | 53~57 | 미달 | 재시공 필요 |
| 5등급 | ≥ 58 | 불량 | — |

### 경량충격음 (Light-weight, Tapping machine)

| 등급 | L_iA,Fmax (dB) | 실적 예시 |
|:---:|:-------------:|:---------|
| **1등급** | ≤ 37 | GS건설+LX하우시스: **27dB** |
| 2등급 | 38~43 | DL이앤씨 D-Silent Floor: 41dB |
| 3등급 | 44~52 | 일반 바닥구조 |

## 한국 층간소음 규제 (2025년 기준)

| 항목 | 내용 |
|:----|:-----|
| 법적 근거 | 공동주택 바닥충격음 차단구조 성능검사 (국토교통부) |
| 적용 대상 | 30가구 이상 공동주택 (2022.8.4 이후) |
| 최소 기준 | 중량충격음 **3등급 (≤52dB)** |
| 검사 표본 | 현행 2% → **5% 이상** 확대 추진 |
| 중간점검 | 전 가구 바닥 공사 전 현장 성능 중간점검 도입 |
| 미달 시 | 보완시공 의무화 |

## 가산비 제도 (경제적 인센티브)

| 등급 | 가산비 | ㎡당 금액 (기본형 217.4만원 기준) |
|:---:|:-----:|:-------------------------------:|
| **1등급** | **+2.0%** | 약 43,000원/㎡ |
| **2등급** | **+1.0%** | 약 21,700원/㎡ |
| 슬래브 250mm↑ | +0.5% | — |
| 슬래브 300mm↑ | +1.0% | (중복 불가) |

→ **1등급 바닥 = ㎡당 4.3만원 추가 분양가 반영 가능**
→ 건설사가 고성능 흡음재에 지불 가능한 추가 비용의 상한선

## 시장 진입 기준선

| 용도 | 필요 α_w | Class | 비고 |
|:----|:-------:|:----:|:----|
| 일반 천장재 | ≥ 0.30 | D | 최소 기준 |
| 사무실/교실 | ≥ 0.60 | C | Approved Document E (UK) |
| 음향 천장재 | ≥ 0.80 | B | 경쟁 제품 수준 |
| 프리미엄 | ≥ 0.90 | A | 공연장급 |
| **폐섬유 흡음재 목표** | **≥ 0.80** | **B** | 100mm 두께, Biot low-freq 보강 |

## 건설사 층간소음 기술 현황

| 건설사 | 기술명 | 성능 |
|:-----|:------|:----:|
| 현대건설 | H-Silent Floor | 1등급 (LH 인증) |
| 삼성물산 | 건식/습식 바닥 | 1등급 이상 |
| GS건설+LX하우시스 | 공동 개발 | 중량 31dB, 경량 27dB (1등급) |
| DL이앤씨 | D-Silent Floor Max | 1등급 (경량+중량) |
| 대우건설 | Smart Silent (350mm) | 중량+경량 1등급 |

## 핵심 인사이트

1. **α_w ≥ 0.80 (Class B) 달성 → 건축 음향 자재 시장 진입 가능**
2. **층간소음 1등급(≤43dB) = 건축비 2% 가산비 → 건설사 경제적 유인**
3. **40-80Hz 저주파는 α_w에 거의 반영 안 됨** — α_w만으로 차별화 어려움
4. **층간소음 44% 사후확인 불합격** = 시장 니즈 확인
