# 건축 음향 규격 — 층간소음 기준 및 흡음 등급

## 개요

한국 층간소음 및 건축 음향 관련 규격의 종합 정리. 제품명세(α_w, L_n,w)와 시장 진입 기준선을 정의.

## ISO 11654:1997 — 흡음 등급 (Sound Absorption Rating)

### α_w (Weighted Sound Absorption Coefficient) 계산 방법

ISO 11654는 단일수치 흡음계수 α_w를 산출하기 위해 **기준곡선 이동법(reference curve fitting)** 을 사용:

1. ISO 354 잔향실법으로 1/3 옥타브 밴드 21개 주파수(100-5000Hz)에서 α 측정
2. 측정된 α_s (practical sound absorption coefficient)를 1/3 옥타브 → 1옥타브로 변환 (각 옥타브 대역의 α_s는 3개 1/3 옥타브 값의 산술평균)
3. **α_p (practical sound absorption coefficient)** 곡선을 다음 기준곡선과 비교:

| 주파수 (Hz) | 기준 α_p 곡선 |
|:----------:|:------------:|
| 125 | 0.10 |
| 250 | 0.15 |
| 500 | 0.20 |
| 1000 | 0.30 |
| 2000 | 0.45 |
| 4000 | 0.55 |

4. 기준곡선을 0.05 간격으로 상하 이동하여, 측정 α_p보다 **평균 0.10 초과**하지 않는 가장 높은 위치 결정
5. 500Hz에서의 기준곡선 값이 **α_w**

### Shape Indicators (L, M, H)

α_w만으로는 흡음재의 주파수 특성을 충분히 표현할 수 없으므로, ISO 11654는 **형상 지시자(shape indicators)** 를 도입:

| Indicator | 의미 | 판정 기준 |
|:---------|:----|:---------|
| **L** (Low) | 저주파 흡음 부족 | 250Hz에서 α_p가 α_w보다 0.25 이상 낮음 |
| **M** (Mid) | 중주파 흡음 우세 | 500Hz 또는 1000Hz에서 α_p가 α_w보다 0.25 이상 높음 |
| **H** (High) | 고주파 흡음 우세 | 2000Hz 또는 4000Hz에서 α_p가 α_w보다 0.25 이상 높음 |

표기 예시: α_w = 0.80 (M, H) → 중·고주파에서 α_w보다 0.25 이상 높은 흡음 성능

### Sound Absorption Classes (ISO 11654 기준)

| Class | α_w 범위 | 용도 예시 |
|:-----|:--------:|:---------|
| **A** | 0.90~1.00 | 공연장, 강당, 녹음실 |
| **B** | 0.80~0.85 | 사무실, 교실, 회의실 |
| **C** | 0.60~0.75 | 복도, 로비, 상업공간 |
| **D** | 0.30~0.55 | 소매점, 경공업 |
| **E** | 0.15~0.25 | 기본 수준 |
| 미분류 | < 0.15 | 흡음성 거의 없음 |

α_w 측정: ISO 354 잔향실, 1/3 옥타브 21개 주파수 평균
저주파 Shape indicator: α_w(N) = 저주파 부족, (M) = 중간 우세, (H) = 고주파 우세
**폐섬유 흡음재 목표: α_w ≥ 0.80 (Class B)** — 100mm 두께 기준

### NRC (Noise Reduction Coefficient) vs α_w

| 항목 | α_w (ISO 11654) | NRC (ASTM C423) |
|:----|:--------------:|:---------------:|
| **계산법** | 기준곡선 이동법 (reference curve fitting) | 250, 500, 1000, 2000Hz 4개 주파수 α의 산술평균 |
| **대역** | 125-4000Hz (6옥타브) | 250-2000Hz (4옥타브) |
| **저주파** | 125Hz 포함 (shape indicator L로 보완) | 미포함 |
| **표기** | α_w + shape indicator (예: 0.80 M) | 단일 숫자 (예: NRC 0.80) |
| **한국** | KS F 3505 (천장재), KS F 4716 (흡음재) | 일부 해외 규격에서 사용 |

> NRC는 계산이 간단하지만 저주파 정보 손실. ISO 11654 α_w + shape indicator가 더 포괄적.

## 국토부 고시 제2023-494호 — 공동주택 바닥충격음 차단구조 인정 및 검사기준

### 개요

- **고시번호:** 국토교통부고시 제2023-494호
- **시행일:** 2023. 8. 28.
- **법적 근거:** 「주택법」 제35조, 「주택건설기준 등에 관한 규정」 제14조의2, 제60조의2~60조의11
- **목적:** 공동주택 바닥충격음 차단성능 측정·평가방법, 성능등급 기준, 성능 검사 기준, 인정기관·검사기관 지정 등

### 용어 정의

| 용어 | 정의 |
|:----|:-----|
| 경량충격음레벨 | KS F ISO 717-2의 "가중 표준화 바닥충격음레벨" |
| 중량충격음레벨 | KS F ISO 717-2의 "A-가중 최대 바닥충격음레벨" |
| 가중 바닥충격음레벨 감쇠량 | KS F 2865 측정 → KS F 2863-1 평가 |
| 완충재 | 충격음 흡수를 위해 바닥구조체 위에 설치하는 재료 |
| 인정기관 | 바닥충격음 차단구조 성능인정 업무 수행 기관 |
| 성능검사기관 | 사용검사 전 성능검사 수행 기관 |

### 바닥충격음 차단성능 등급 기준 (별표 1)

#### 중량충격음 (Heavy-weight, Bang machine / Rubber ball)

| 등급 | L_iA,Fmax (dB) | 평가 | 시장 의미 |
|:---:|:-------------:|:----:|:--------|
| **1등급** | ≤ 43 | 우수 | 프리미엄 (건축비 +2%) |
| **2등급** | 44~48 | 양호 | 고급 아파트 |
| **3등급** | 49~52 | **법적 기준** | 최소 요구사항 |
| **4등급 (벽식)** | 53~57 | 미달 | 재시공 필요 |
| **4등급 (라멘)** | 별도 표기 | — | 슬래브 160mm↑ 조건 |

#### 경량충격음 (Light-weight, Tapping machine)

| 등급 | L_iA,Fmax (dB) | 실적 예시 |
|:---:|:-------------:|:---------|
| **1등급** | ≤ 37 | GS건설+LX하우시스: **27dB** |
| **2등급** | 38~43 | DL이앤씨 D-Silent Floor: 41dB |
| **3등급** | 44~52 | 일반 바닥구조 |
| **4등급** | ≥ 53 | 미달 |

### 구조 방식별 적용 기준

| 구조 방식 | 성능인정 필요 여부 | 특이사항 |
|:---------|:---------------:|:--------|
| 벽식구조 | ✅ 필수 | — |
| 무량판구조 | ✅ 필수 | — |
| 혼합구조 | ✅ 필수 | — |
| 라멘구조 | 슬래브 160mm↑ 시 인정 가능 | 가중 바닥충격음레벨 감쇠량 ≥ 13dB 또는 20mm↑ 완충재 필요 |

- 벽식구조 인정 → 무량판구조·혼합구조에도 적용 가능 (동일 슬래브 두께·형상·온돌층 조건)
- 바닥충격음 차단구조 = 슬래브 포함 상부 구성체 (바닥마감재는 제외, 단 신청 시 포함 가능)

### 사후확인제도 (성능검사, 제4장)

**제28조(성능검사 대상 및 측정 세대 선정)**

- 대상: 벽식구조, 무량판구조, 혼합구조, 라멘구조 등 주택에 적용된 모든 바닥구조
- 표본 수: 사업계획승인 단지 유형별 세대수의 **2%** (소수점 올림)
- 유형 분류 기준:
  1. 전용면적: 60m² 이하 / 60-85m² / 85m² 초과
  2. 층수: 10층 이하 / 10-30층 / 30층 초과
- 선정 방법: 유형별 **무작위 추출**

**제30조(측정결과 평가)**

- 측정결과 = 산술평균값
- 판단기준 = 별표 1의 바닥충격음 차단성능 등급기준
- 성능검사기준 = 주택건설기준 제14조의2제1항제2호

### 검사기관 지정 체계

**제5조(인정기관 지정기준)**

인정기관 자격 요건:
1. 법인으로서 바닥충격음 차단구조 성능등급 인정업무 수행 조직 보유
2. 공정·신속한 인정업무 수행 체계
3. 설계·공사감리·건설·부동산업·건축자재 제조·공급·유통업 **미해당**
4. 바닥충격음 차단구조 관련 연구실적 및 유사업무 수행 능력

**제31조(성능검사기관 지정)**
- 성능검사기관 지정은 제6조(인정기관 지정 절차)를 준용
- 업무: 공인시험기관 선정·관리·감독, 검사 신청 접수·결과 통보, 대상 세대 선정·검사, 데이터 관리·분석

### 인정 유효기간 및 연장

| 항목 | 내용 |
|:----|:-----|
| 최초 인정 유효기간 | **5년** (공고일 기준) |
| 연장 신청 | 만료 6개월 전 ~ 만료 전 |

### 품질관리 체계

**제19조(품질관리 상태 확인점검)**
- 매년 **2회 이상** 공장품질관리 확인점검 실시
- 전년도 지적 없음 시 면제 가능
- 현장 시공실적 없음 시 3년간 점검 제외
- 인정기관 합동 점검 가능

**제20조(인정자의 자체품질관리)**
1. 구성재료·원재료 검사
2. 제조공정 중간검사 및 공정관리
3. 제품검사 및 제조설비 유지관리
4. 제품생산·판매실적·시공현장 상세내역 기록·보존

| 동탄성계수 + 손실계수 | KS F 2868 | 하중판 거치 48시간 후 측정 |

### 측정 및 평가 방법 (제26-27조)

- 측정 표준: **KS F ISO 16283-2** (현장 바닥충격음 차단성능 측정)
- 평가 표준: **KS F ISO 717-2** (경량=가중 표준화, 중량=A-가중 최대)
- 마이크로폰 높이: 바닥으로부터 **1.2m**
- 측정 지점: 중앙 1개소 + 벽면 0.75m(14m² 미만 시 0.5m) 떨어진 4개소
- 시험 조건: 음원실·수음실 바닥면적 20m² 미만 + 20m² 이상 각각 2곳

| **1등급** | **+2.0%** | 약 43,000원/㎡ |
| **2등급** | **+1.0%** | 약 21,700원/㎡ |

## 핵심 인사이트

1. **α_w ≥ 0.80 (Class B) 달성 → 건축 음향 자재 시장 진입 가능**
2. **층간소음 1등급(≤43dB) = 건축비 2% 가산비 → 건설사 경제적 유인**
3. **40-80Hz 저주파는 α_w에 거의 반영 안 됨** — α_w만으로 차별화 어려움
4. **층간소음 44% 사후확인 불합격** = 시장 니즈 확인
5. ISO 11654 기준곡선법은 α_w 산출에 6개 옥타브(125-4000Hz) 사용, shape indicator(L/M/H)로 주파수 특성 보완
6. 국토부 고시 2023-494호는 라멘구조를 별도 등급 체계로 분리, 사후확인 표본 2% 유지 (5% 확대 추진 중)
