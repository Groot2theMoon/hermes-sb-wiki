# Autoencoder-Based ROM

Autoencoder-based reduced order models compress high-dimensional simulation data into a low-dimensional latent space using autoencoders, then evolve the latent dynamics efficiently. Related to [[neural-odes|neural ODEs]] and [[pseudo-differential-neural-operator|PDNO]].
