# Waste-Fiber Acoustic Absorber — 시장 진입 전략

## 한국 층간소음 규제 현황

### 사후확인제도 (2022년 8월 도입)

| 연도 | 기준 (경량/중량) | 특징 |
|:----|:--------------|:------|
| 2022 (도입) | 경량 58dB / 중량 50dB (4등급) | 최소 기준, 샘플 검사 2~5% |
| **2024 (강화)** | **경량 49dB / 중량 49dB** | 등급 기준 대폭 강화 |
| LH 자체 기준 | 경량 2등급, 중량 3등급 | 공공주택에 더 엄격한 기준 적용 |

### 사후확인제도의 시장 파급 효과

- 2024년 검사 결과: 9개 단지 중 **4개 단지(44%) 기준 미달**[^aptn]
- 불합격 시: 재시공 비용, 입주 지연, 민원 발생
- 건설사 입장: 기존 바닥구조(스티로폼/EVA 완충재)로 49dB 달성이 어려워짐[^rcnews]
- **→ 기존 솔루션으로 안 되는 시점 = 신규 기술 진입의 최적기**

## 폐섬유 처리 비용 및 제도

### 폐섬유 발생 현황 (한국)

| 항목 | 수치 | 출처 |
|:----|:----|:----|
| 연간 폐의류·폐섬유 발생량 | **약 80만 톤** | KEI 한국환경연구원 |
| 재활용률 | **38%** (실질적 물질 재활용은 4.7%) | 2023 환경부 자료 |
| 주 처리 방식 | 중고의류 수출 25.7%, 에너지 회수 5.9%, 물질재활용 4.7% | KEI 보고서 |
| 현실 | "한국은 단 한 번도 폐의류를 재활용한 적이 없다" — 대부분 해외 수출 후 현지 소각·매립 | KEI 보고서 |

### 폐기물 처리 비용 (2024~2025 기준)

| 구분 | 비용 (톤당) | 비고 |
|:----|:----------|:----|
| 사업장 일반폐기물 소각 | **약 10~15만원** (민간 소각장 기준 ~10.8만원) | 에너지 회수 시 감면 |
| 사업장 일반폐기물 매립 | 약 8~12만원 | 토지 사용 + 환경 부담금 |
| 폐합성수지(섬유) 재활용 | 5~10만원 | 분리 상태에 따라 변동 |
| 운반비 별도 | 거리/업체별 상이 | |
| 폐기물처분부담금 | 소각 kg당 10~25원, 매립 kg당 10~15원 | 환경부 고시 (정부 추가 부담금)[^keco] |

### 핵심 인사이트: "돈 받고 폐섬유 가져가기"

폐섬유 배출업체의 입장:
1. **폐기물 처리비용 부담**: 톤당 10~25만원 + 운반비
2. **폐기물처분부담금**: 소각/매립 시 kg당 10~25원 추가 부담
3. **EPR 도입 임박**: 섬유 생산자책임재활용제도 도입 예정 → 생산자가 폐기물 관리비용 직접 부담[^social]

→ **"처리 비용을 내는 대신, 무료로 가져가거나 일부 보전해주는 조건" 협상 가능**

### 추정 경제성 (톤당 기준)

| 항목 | 일반 소각 처리 | 당신에게 공급 |
|:----|:------------|:----------|
| 배출업체 부담 | -15만원 (처리비) | **0원 (무상 인계)** |
| | -2.5만원 (처분부담금 25원/kg) | |
| 당신 입장 | — | 원재료비 **0원** (또는 수취) |
| 총 배출업체 절감 | — | **17.5만원/톤 절감** |

### 경쟁 구도 (폐섬유 흡음재 분야)

| 경쟁자 | 제품/방식 | 특징 |
|:------|:--------|:----|
| **오슬로(Osllo)** | 슬로우넬 — 폐섬유 고온 압축 패널 | 흡음+방염+충격흡수, 내장재/데크 등 |
| 기존 GR 인증 제품 | 재활용 면섬유흡음재 (GR K 0012) | 이미 GR 인증 품목 있음 |
| LG하우시스 / KCC | 기존 흡음재 (유리/암면 계열) | Virgin 원료, 고가 |
| **당신의 차별점** | **DMN 기반 최적 설계 + 60Hz 저주파 타겟 + MaaS** | 설계 정밀도 + 맞춤형 생산 |

## GR 인증 전략

### 이미 존재하는 GR 품목 (폐섬유 분야)

| 품목코드 | 제품명 |
|:-------|:------|
| GR K 0001 | 재활용 폴리에스테르 스테이플 섬유 |
| **GR K 0004** | **재활용 섬유흡음재** ⬅ 직행 가능 |
| GR K 0011 | 재활용 나일론 섬유 |
| **GR K 0012** | **재활용 면섬유흡음재** ⬅ 직행 가능 |
| GR K 0013 | 폐원면 재활용 면양말 |

→ **GR K 0004 또는 GR K 0012**에 해당하는 제품으로 인증 취득 → 조달청/공공기관 우선 구매 대상 진입

### GR 인증의 효과

| 효과 | 내용 |
|:----|:----|
| **공공기관 의무구매** | GR 인증 제품은 공공기관 우선 구매 대상 (법적 의무) |
| **조달청 가점** | 2단계 경쟁 시 환경 분야 가점 부여 |
| **LH 특혜** | LH 녹색제품 구매 연간 1.58조원 규모[^lh-esg] |
| **ESG 홍보 효과** | 건설사의 ESG 보고서에 반영 가능 |

## 진입 로드맵

### Phase 0 (석사 1년차): 기술 검증
- POSTECH DSLab에서 DMN 기반 흡음재 설계 프레임워크 구축
- 폐섬유 시편 수집 + impedance tube 측정 (POSTECH 음향 실험실)
- DMN 예측 vs 실제 측정값 비교 논문

### Phase 1 (석사 2년차): LH R&D 과제 수주
- LH 토지주택연구원 공동 연구 제안
- "폐섬유 기반 저주파 층간소음 저감 흡음재 개발"
- 연구비: 2~5억원 (1~2년)
- **GR 인증 병행 추진** (규격 GR K 0004 또는 K 0012)

### Phase 2 (박사 과정): 시범 적용 + MaaS
- LH 임대아파트 1~2개 단지 시범 적용[^rcnews]
- 사후확인제도 성능 검증 (실제 현장 측정)
- 생산 거점 확보 (대구 섬유산업단지 인근)

### Phase 3 (창업): 스핀오프
- 연구실 창업 (딥테크)
- MaaS (Manufacture as a Service) 모델
- 대구/수도권 거점 생산

## 조달/환경 관련 법령

| 법률/제도 | 주요 내용 | 관련성 |
|:--------|:--------|:------|
| **주택법** 제41조 | 바닥충격음 차단성능 사전인정 + 사후확인 | 흡음재 성능 기준 |
| **자원순환기본법** | 폐기물처분부담금 부과[^keco] | 폐섬유 처리 비용 |

| **탄소중립기본법** | 온실가스 감축 의무 | 재활용 제품 탄소배출 저감 인정 |
