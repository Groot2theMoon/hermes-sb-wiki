# Active Inference AI Systems for Scientific Discovery

Friston의 **Free Energy Principle**을 실제 AI 과학 발견 시스템으로 구현한 프레임워크. Active inference = perception + action = surprise 최소화.

## 핵심 아이디어

- **Long-lived research memories:** causal self-models을 유지
- **Active inference:** 세계 모델 예측 오차를 행동으로 줄임
- **Scientific discovery:** 가설 생성 → 실험 설계 → 결과를 모델 업데이트

## FEP과의 연결

이 논문은 [[free-energy-principle]]의 실용적 AI 구현체:
- Perception = Bayesian inference on scientific data
- Action = experiment design to minimize expected free energy
- Learning = model update via prediction error minimization
