# Information Thermodynamics — Sagawa-Ueda Relations

Sagawa & Ueda가 정립한 정보-열역학 이론. **Landauer 원리 ↔ Friston FEP**를 연결하는 핵심 가교.

## 핵심 수식: Sagawa-Ueda Relation (2008)

```
⟨exp(-σ - I)⟩ = 1

where: σ = dissipated entropy
       I = mutual info between system state and measurement
```

→ Maxwell's Demon이 information을 활용하면 **Landauer 한계 이상의 work 추출 가능**

## FEP와의 연결

Sagawa-Ueda 피드백 구조:
1. Measurement = 정보 획득 (↔ FEP의 perception)
2. Feedback = 정보 활용하여 work 추출 (↔ FEP의 action)
3. Net result: information으로 entropy 감소 가능
