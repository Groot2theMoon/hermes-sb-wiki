# Engram: Scalable Sparse Memory for LLMs

- **참고자료:** Transformer의 attention window 제약을 memory lookup으로 보완
- [[transformer]]의 context window 한계를 외부 메모리로 해결하는 접근
- [[transformer]] — Attention 기반 LLM
