# Philipp Becker

Researcher, Autonomous Learning Robots (ALR), Karlsruhe Institute of Technology (KIT). RKN (Recurrent Kalman Network) 창시자.

## 주요 연구
- **RKN (ICML 2019):** Kalman filter + deep encoder-decoder 통합 probabilistic recurrent architecture. Factorized latent state + locally linear transition. LSTM 대비 state estimation 우위.
- **ac-RKN (CoRL 2020):** RKN을 action-conditional prediction/inverse dynamics로 확장 (Shaj et al.)
- Factorized inference in high-dimensional deep feature spaces
- Probabilistic time-series modeling, uncertainty-aware learning
