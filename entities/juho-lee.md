# Juho Lee

## 개요

Juho Lee는 Oxford University Department of Statistics와 AITRICS에서 활동하는 연구자로, **attention 메커니즘 기반 집합 구조 데이터 처리**, **확률적 머신러닝**, **베이지안 딥러닝** 분야에서 활동하고 있다. Set Transformer의 제1저자이자 교신저자.

## 주요 연구

- **Set Transformer** (2019, ICML) — [[set-transformer]]: Attention 기반 permutation-invariant 신경망 아키텍처
- Induced Set Attention Block (ISAB) — m개의 inducing point로 self-attention 복잡도를 O(n²)에서 O(nm)으로 감소
- Pooling by Multihead Attention (PMA) — 학습 가능한 seed 기반 attention pooling
- AITRICS에서의 확률적 모델링 및 explainable AI 연구
