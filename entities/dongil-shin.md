## 연구 분야

데이터 기반 방법론과 물리 기반 모델을 결합한 고체역학 문제 해결:

- **Deep Material Network (DMN):** 물리 기반 surrogate model — multiscale/multiphysics 재료 설계
- **차원 축소 모델링 (Reduced-order modeling):** 실시간 시뮬레이션
- **구조 최적화 (Structural optimization):** 데이터 기반 설계
- **불확실성 정량화 (Uncertainty Quantification):** Bayesian ML 기반 센서 설계
- **나노기계 공진기 (Nanomechanical resonators):** 초저손실 구조 설계

## 주요 연구 성과

1. **데이터 기반 박판 구조 해석/모델링:** 성능 향상을 위한 축소 요소 개발
2. **광기계 센서 (Optomechanical sensors) 기록 경신:** Bayesian ML로 설계 프로세스 가이드
3. **Deep Material Network:** 다중 스케일/다중 물리 기계 설계를 위한 PML
