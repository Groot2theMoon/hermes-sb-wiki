# acoustipy

- **GitHub:** https://github.com/jakep72/acoustipy
- **Author:** jakep72
- **Stars:** ⭐ 9 | **License:** MIT
- **Language:** Python

acoustipy는 Allard & Atalla (2009)의 Transfer Matrix Method(TMM)와 Johnson-Champoux-Allard(JCA) 모델을 Python으로 구현한 오픈소스 패키지. JCA 5-parameter → 다층 흡음 구조의 흡음계수 α(f) 계산을 지원하며, 역방향 물성 추정(inverse optimization) 기능도 포함.
