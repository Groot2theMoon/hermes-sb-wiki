# José C. Príncipe

Distinguished Professor, University of Florida. Director, Computational NeuroEngineering Laboratory (CNEL). Life Fellow, IEEE.

## 연구 분야
- Computational neuroscience
- Kernel adaptive filtering (KAF) and kernel methods
- Adaptive signal processing
- Brain-machine interfaces
- Nonlinear dynamical systems

## 주요 기여

### Kernel Adaptive Filtering (KAF) 프레임워크
CNEL에서 20년 이상 kernel adaptive filtering 이론을 발전시켜 옴. KAARMA (kernel adaptive autoregressive-moving-average)부터 FBF, expFBF까지의 연구 계보 확립.

### Kernel Operator-Theoretic Bayesian Filter (2024)
- arXiv:2411.00198 — with Kan Li
- RKHS + Koopman operator 융합을 통한 nonlinear Bayesian filtering
