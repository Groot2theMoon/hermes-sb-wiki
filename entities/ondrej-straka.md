# Ondřej Straka

Associate Professor at the University of West Bohemia (UWB), Department of Cybernetics. Research focuses on **nonlinear filtering**, sigma-point methods, adaptive Kalman filtering, and stochastic systems. Long-time collaborator with Jindřich Duník and Miroslav Šimandl.

## 연구 분야
- Nonlinear state estimation and Bayesian filtering
- Sigma-point methods (UKF, sigma-point set rotation)
- Adaptive Kalman filtering (scaling parameter adaptation)
- Stochastic integration filters (SIF)
- Point-mass filters for non-Gaussian systems

## 주요 기여
- **UKF scaling parameter adaptive setting** (with [[jindrich-dunik]] & [[miroslav-simandl]], IEEE TAC 2012)
- **Sigma-point set rotation** — Analysis and adaptation of sigma-point configurations in UKF (IFAC 2014)
- **Stochastic integration filter (SIF)** — Novel quadrature-based Gaussian filtering framework
- **Truncated randomized UKF** — Handling interval-constrained state estimation
- **Improved calibration of numerical integration error** in sigma-point filters (with T. Karvonen, S. Särkkä, IEEE TAC 2021)
- **Student-t Process Quadratures** — Heavy-tailed noise filtering (FUSION 2017, Best Student Paper Runner-up)
