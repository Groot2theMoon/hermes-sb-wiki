# Amit Levy

Researcher affiliated with the University of Haifa, Hatter Department of Marine Technologies. Research focuses on **nonlinear filtering**, unscented Kalman filter variants, and navigation systems. Co-author of MS-UKF (Multi-Scaled Unscented Kalman Filter) with Itzik Klein.

## 연구 분야
- Nonlinear state estimation and Kalman filtering
- Adaptive and multi-scaled unscented Kalman filters
- Inertial navigation systems (INS)
- Machine learning for navigation
- Robot localization

## 주요 기여
- **Multi-Scaled UKF (MS-UKF)** — Per-state scaling parameter for UKF (with [[itzik-klein]], arXiv:2604.04792, 2026)
- **Adaptive Neural UKF** — Neural network-aided adaptive UKF for navigation (IEEE Trans. Intelligent Vehicles, 2026)
- **UKF with nonlinear propagation model** — Improved navigation accuracy using nonlinear error-state dynamics
