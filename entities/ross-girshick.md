# Ross Girshick

**R-CNN / Fast R-CNN 창시자, YOLO 공동 창시자.** Meta AI (Facebook AI Research) 연구자로, 현대 객체 탐지(object detection)의 **region-based 접근법**을 개척했다.

## 주요 기여

### R-CNN 계열
- **R-CNN (2014)**: Region proposal + CNN 분류로 객체 탐지의 CNN 기반 접근을 최초로 성공시킴
- **Fast R-CNN (2015)**: RoI pooling + multi-task loss 도입으로 속도와 정확도 혁신
- **Faster R-CNN (2016, Ren et al.)**: Region Proposal Network (RPN) 도입에 기여

### YOLO
**Redmon**, Divvala, **Girshick**, Farhadi (2015, CVPR 2016)가 공동 제안한 [[yolo-object-detection]]는 실시간 객체 탐지의 패러다임을 바꾼 single-shot detector이다.

### 연구 분야
- Object detection and instance segmentation
- Region-based convolutional networks
- Visual recognition and representation learning
