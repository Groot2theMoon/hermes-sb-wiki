# Paris Perdikaris

**PINN (Physics-Informed Neural Networks) 공동 창시자.** University of Pennsylvania (UPenn) 교수로, [[physics-informed-neural-networks]] 및 [[physics-constrained-surrogate]] 연구의 핵심 인물.

## 주요 기여

### Physics-Informed Neural Networks (PINN)
Raissi, **Perdikaris**, Karniadakis (2019)가 공동 제안한 [[physics-informed-neural-networks]]는 PDE residual을 손실 함수에 포함하여 데이터 없이 물리 법칙만으로 해를 근사하는 방법론이다.

### Physics-Constrained Deep Learning
Zhu, Zabaras, Koutsourelakis, **Perdikaris** (2019)의 [[physics-constrained-surrogate]]는 레이블 데이터 없이 물리 법칙만으로 고차원 Surrogate Model을 학습하는 방법을 제시했다.

### PINN Failure Modes
Wang, Yu, **Perdikaris** (2021)의 [[pinn-failure-modes]]는 NTK (Neural Tangent Kernel) 관점에서 PINN 학습 실패의 근본 원인을 규명했다.

### 연구 분야
- Physics-informed deep learning
- Scientific machine learning (SciML)
- Surrogate modeling for PDEs
- Data-driven dynamical systems
