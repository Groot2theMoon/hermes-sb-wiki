# Mattia Zorzi

Professor at **University of Padova** (Dipartimento di Ingegneria dell'Informazione). Expert in **robust filtering, model predictive control, and information-theoretic estimation**.

## 주요 기여

- **Robust Kalman filtering under model perturbations** (*IEEE TAC* 2016) — KL divergence-based ambiguity set
- **Robust sigma point Kalman filtering** (with [[shenglun-yi|Shenglun Yi]], 2025) — Extension to nonlinear systems via sigma point ambiguity set center approximation
- **Robust least-squares estimation with relative entropy constraint** (with Levy, *IEEE TIT* 2004)
- **Coupling of MPC and robust Kalman filtering** (*IET CTA* 2018)
- **Learning tuned liquid damper dynamics via robust EKF** (ACC 2021)
