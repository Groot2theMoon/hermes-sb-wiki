# Bernhard Schölkopf

## 개요

Bernhard Schölkopf는 **Max Planck Institute for Intelligent Systems (Tübingen)**의 소장. 커널 방법(kernel methods), SVM, kernel PCA의 공동 창시자로서 ML 역사에 지대한 공헌. Klaus-Robert Müller의 오랜 공동 연구자.

## 주요 기여

### Kernel PCA (Schölkopf, Smola, Müller 1998)
- 기존 PCA를 커널 trick으로 확장하여 **비선형 차원 축소** 가능하게 함
- 대조군: 선형 PCA와 대비되는 비선형 차원 축소의 표준

### SVM과 커널 방법 (Müller, Mika, Rätsch, Tsuda, Schölkopf 2001)
- SVMs, KFD, KPCA를 포함한 커널 학습 방법의 종합 서베이
- **~5,000회 인용**, 커널 방법 분야의 고전

### Causal Inference (후기 연구)
- Julia Rohrer, Jonas Peters 등과 인과 추론(causal inference) 연구
- ML에 인과 관계를 통합하는 프레임워크 개발
