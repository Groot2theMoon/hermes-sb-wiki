# 삼겹살 양송이 김치볶음 덮밥 레시피

- 작성일: 2026-04-29
- 출처: 승원님 자택 요리 (AI 어시스턴트 레시피)

## 재료 (1인분)

| 재료 | 분량 | 비고 |
|------|------|------|
| 돼지 삼겹살 | 180g | 먹기 좋게 썰기 |
| 신김치 | 1.5컵 (~200g) | 가위로 잘게 |
| 양송이버섯 | 4~5개 | 얇게 슬라이스 |
| 양파 | 1/2개 | 채 썰기 |
| 대파 | 1대 | 어슷썰기 |
| 굴소스 | 1/2T | |
| 간장 | 1T | |
| 고추장 | 1/2T | |
| 고춧가루 | 1T | |
| 다진마늘 | 1T | |
| 설탕 or 꿀 | 1t | |
| 참기름 | 1t | 마무리용 |
| 밥 | 1공기 | |
| 계란 | 1개 | 프라이 (옵션) |
| 통깨 | 약간 | 솔솔 |

## 조리 순서

### 1단계: 재료 손질
- 삼겹살은 2~3cm 폭으로 썰기
- 양송이버섯은 3~4mm 두께로 슬라이스
- 양파는 채 썰고, 대파는 어슷썰기
- 김치는 가위로 잘게 자르기

### 2단계: 삼겹살 굽기
- 팬에 기름 약간만 두르고 중강불
- 삼겹살 넣고 노릇노릇하게 구워주기 (약 5분)
- 겉이 바삭해지면 한쪽으로 밀기

### 3단계: 양송이버섯 볶기
- 같은 팬, 삼겹살 기름에 양송이버섯 넣기
- 소금 약간 뿌려가며 2분간 볶기
- 버섯이 노릇해지면 삼겹살과 섞기

### 4단계: 김치 넣고 볶기
- 김치를 팬에 넣고 2~3분 볶기
- 김치가 익으면서 투명해질 때까지

### 5단계: 양념 넣기
- 굴소스 1/2T + 간장 1T + 고추장 1/2T
- 고춧가루 1T + 다진마늘 1T + 설탕/꿀 1t
- 모든 재료 잘 섞이도록 2분간 볶기

### 6단계: 양파+대파 투입
- 채 썬 양파와 대파 넣고
- 1~2분 더 볶기 (아삭함 남도록)

### 7단계: 마무리
- 불 끄고 참기름 1t 둘러주기
- 통깨 솔솔~

### 8단계: 계란 프라이 (옵션)
- 별도 팬에 계란 프라이 하나
- 노른자 터뜨려 비비면 감동

## 플레이팅
밥 → 삼겹살 김치볶음 → 계란 프라이 → 통깨

## 팁
- 김치가 짜면 밥 더 넣어서 볶음밥으로 업그레이드 가능
- 김치가 너무 시면 설탕 1/2t 추가
- 기름이 너무 많으면 중간에 키친타월로 한 번 닦아주기
