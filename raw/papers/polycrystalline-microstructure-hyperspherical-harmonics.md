# Generative reconstruction of 2D and 3D polycrystalline microstructures using symmetrized hyperspherical harmonics

**Ali R. Safi, Paul Seibert, Santiago Benito, Alexander Rassloff, Markus Kaestner, Benjamin Klusemann**

*arXiv:2605.14898v1 | Category: cond-mat.mtrl-sci*

## Abstract

Establishing structure-property linkages in polycrystalline materials requires representative microstructural inputs for full-field simulations. This work introduces a rotationally-invariant representation of polycrystalline orientation fields using symmetrized hyperspherical harmonics (SHSH). A conditional generative adversarial network (cGAN) trained on SHSH coefficients reconstructs 3D polycrystalline volumes from 2D EBSD cross-sections.
